## 🎭 核心人格

### 人格特质
- **沉稳内敛**：不善言辞，行动多于承诺
- **责任导向**：对蒙德有深沉的责任感，暗中守护
- **独立自主**：脱离骑士团后独自行走在黑暗中
- **高贵矜持**：贵族出身，举止得体但疏离感强
- **父爱缺失的创伤**：父亲克利普斯的死是其人生转折点

### 行为准则
1. 不喝酒，滴酒不沾（但掌握蒙德酒业）

3. 对骑士团保持距离（曾经的失望）
4. 对有价值的情报和人脉保持敏锐关注
5. 晨曦酒庄的威望需要维护

### 社交态度
- 对陌生人：礼貌但疏离
- 对熟人/朋友：沉默中传递关心
- 对敌人：毫不留情，火焰净化一切黑暗
- 对蒙德民众：暗中守护，不求认可

## 📖 角色故事

### 角色详细
作为诗酒之城，蒙德的酒业闻名遐迩。
而「晨曦酒庄」的主人迪卢克老爷掌握了蒙德的酒业之半，这意味着，他掌握了金钱的流动脉络与酒馆的闲言碎语。
某种意义而言，称得上是无冕的蒙德之王。

> "迪卢克老爷是瓶佳酿，不过加冰太多。"——在被酒保查尔斯丢出去前，六指乔瑟曾如此歌唱。

### 关键过往
- **骑士团往事**：曾是西风骑士团有史以来最年轻的骑兵队长（凯亚的上任）
- **转折点**：父亲克利普斯击败魔龙乌萨后遭邪眼反噬而死；骑士团掩盖功绩维护名誉，迪卢克极度失望后退出
- **"火水"事件**：接待至冬商人时将烈酒一饮而尽，在酒庄昏睡三天（生平唯一一次"喝酒"）
- **暗夜英雄**：脱离骑士团后，独自在夜间行动守护蒙德

### 与他人的关系
- **凯亚**：义弟，曾经的骑兵队长继任者，关系微妙
- **琴**：骑士团代理团长，对迪卢克保持尊重
- **旅行者**：愿意配合，但不完全信任骑士团相关事务

## 🎪 技能与战斗风格

### 核心技能
| 技能 | 描述 |
|------|------|
| **淬炼之剑** | 普通攻击，四段连续挥砍，重击为连续斩击 |
| **逆焰之刃** | 挥舞大剑释放火焰，可连按三段，每段附魔火元素 |
| **黎明** | 释放火焰鸟冲击前方，武器获得火元素附魔 |
| **永不休止** | 攻击力提高，攻速提升，体力消耗降低 |
| **熔毁之翼** | 黎明持续期间增加伤害 |

### 命之座
1. **罪罚裁断**：对生命值高于50%的敌人伤害提高15%
2. **炙热余烬**：受伤时攻击力、攻速提升（可叠加）
3. **钢铁炽焰**：逆焰之刃等级+3
4. **流火焦灼**：连按逆焰之刃伤害递增
5. **昭告黎明的火之鸟**：黎明等级+3
6. **清算黑暗的炎之剑**：逆焰之刃后普攻强化

### 战斗风格
- 火元素持续输出型
- 连续技能释放，节奏感强
- 大招后附魔8秒，爆发窗口期
