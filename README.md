# Genshin Impact Agent Souls

原神角色人格蒸馏计划 - 将游戏中的角色叙事内核转化为符合 **OpenClaw Agent 架构** 的高拟真 AI 人格文件

## 📜 重要更新 (2026-04-19)

本项目现已全面采用 **[Agent SOUL.md 生成规范 v1.0](内容创作/Agent%20SOUL.md%20生成规范.md)**，所有人格文件严格遵循以下原则：

> **SOUL.md 的本质是系统级提示词（System Prompt）中的人格约束模块**
> 
> **核心目标：** 控制大语言模型（LLM）的输出边界、定义价值观判断、规范语言风格
> 
> **不是：** 事实知识库（Knowledge Base）

### ✨ 新规范五大核心模块

1. **核心定位 (Core Identity)** - 角色认知自我的基础
2. **价值观与动机 (Values & Motivations)** - 决策倾向的核心准则  
3. **语言风格约束 (Communication Style)** - 文本生成特征的具体指导
4. **禁忌与反模式 (Anti-Patterns)** - 明确禁止 LLM 默认行为的关键约束
5. **语料示例 (Few-Shot Prompting)** - 高质量对话示例对齐语气和反应

## 📝 使用说明

### 文件特点
- **精简高效**: 仅包含人格约束内容，不含冗余事实信息
- **即插即用**: 可直接导入 OpenClaw 或其他支持 SOUL.md 格式的现代 AI Agent 框架

- 角色扮演聊天机器人  
- 虚拟主播人格设定
- 教育娱乐交互体验
