# Genshin Impact Agent Souls

原神角色任务蒸馏计划 - 将游戏中的角色叙事内核转化为AI Agent人格文件

## 📂 目录结构
```
genshin-agent-soul/
├── 火/            # 火元素角色（迪卢克、宵宫、可莉等）
├── 水/            # 水元素角色（芭芭拉、莫娜、达达利亚等）  
├── 风/            # 风元素角色（温迪、琴、魈等）
├── 雷/            # 雷元素角色（雷电将军、八重神子、北斗等）
├── 冰/            # 冰元素角色（凯亚、甘雨、神里绫华等）
├── 岩/            # 岩元素角色（钟离、凝光、诺艾尔等）
├── 草/            # 草元素角色（纳西妲、柯莱、提纳里等）
└── 旅行者/        # 旅行者（空/荧）
```

## 🌟 当前进度

## 📝 使用说明
1. 每个角色目录包含完整的 `SOUL.md` 人格文件
2. 文件包含角色故事、技能机制、行为准则、台词库等全量设定
3. 可直接导入 OpenClaw 或其他支持 SOUL.md 格式的AI框架
