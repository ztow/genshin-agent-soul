## 📝 使用说明

### 文件特点
- **精简高效**: 仅包含人格约束内容，不含冗余事实信息
- **即插即用**: 可直接导入支持 SOUL.md 格式的现代 AI Agent 框架
- **高拟真度**: 通过明确的禁忌约束和对话范例，有效控制 LLM 输出边界

### 典型应用场景
- 游戏 NPC 对话系统
- 角色扮演聊天机器人  
- 虚拟主播人格设定
- 教育娱乐交互体验
